## 使用示例

### 国家级采集
```python
from scripts.fetcher import GovDocFetcher

fetcher = GovDocFetcher()
items = fetcher.fetch_list('ndrc', 'national')
print(f"发改委: {len(items)} 条")
```

### 省级采集
```python
from scripts.fetcher import GovDocFetcher

fetcher = GovDocFetcher()
items = fetcher.fetch_list('guangdong', 'provincial')
print(f"广东省: {len(items)} 条")
```
